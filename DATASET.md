# 📊 Data Source Documentation

This repository utilizes a large-scale, multi-million-row global sports dataset to evaluate athlete performance and operational tracking metrics.

## 🔗 Original Source Link
* **Dataset Sourced From:** Kaggle
* **Direct Download Link:** https://www.kaggle.com/datasets/aiaiaidavid/the-big-dataset-of-ultra-marathon-running
* **Maintained By:** David 

## 📦 Dataset Properties & Footprint
* **Raw File Format:** Flat CSV (Comma-Separated Values)
* **Compressed File Size:** ~258 MB (.zip file archive)
* **Uncompressed Storage footprint:** ~1.62 GB
* **Total Entry Scale:** 7,461,226 rows × 13 column variables
